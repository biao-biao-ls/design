## 📋 文档概述

**项目名称**: KNZN 硬件学习网站 - 社区中心页面  
**路由**: `/hub`  
**文档版本**: v3.0

**审核状态**: ✅ 最终确定版本  
**文档类型**: 完整设计规范

## 🎯 核心定位

**社区中心 (Community Hub)** 是 KNZN 平台的技术交流和虚拟资产分享中心，用户在这里可以：

- 📝 分享硬件项目、设计心得、调试技巧
- 🔍 寻求技术帮助，解答他人疑惑
- 🏆 展示成就、参与代码审查、获得认可
- 💻 分享可运行的 Wokwi 仿真项目
- 📢 发布技术讨论、学习心得、最佳实践
- 🎬 嵌入实时仿真演示，让项目"活"起来

**核心理念**：专注技术交流，打造硬件学习者的知识分享社区。

## 🎯 MVP 核心功能（专注技术交流）

### 核心理念：专注技术交流，打造硬件学习者的知识分享社区

基于独立开发者的现实约束，MVP 阶段只保留与核心业务（Wokwi 仿真学习）结合最紧密的功能：

**保留的核心功能**：
- 📝 **基础 Feed**：用户分享项目和求助
- 🔍 **代码审查广场**：这是用户最痛点的功能
- 🎬 **嵌入仿真功能**：与 Wokwi 深度集成的核心差异化

**MVP 阶段移除的功能**：
- ❌ 团购 (Group Buy) - 去电商化
- ❌ 文章投稿 (Articles) - 避免内容管理复杂度
- ❌ 复杂的用户关注系统
- ❌ 私信功能
- ❌ 活动中心（线下活动组织）

**原因**：社区需要极高的"热启动"成本。没人发帖的社区比没有社区更糟糕。让用户聚焦在"我卡关了，求大神帮我看看代码"这一件事上。

## 🎬 核心功能模块

### 1️⃣ 🎬 富媒体调试沙箱 (Embedded Simulator)

**让项目"活"起来**：社区帖子里的项目应该是可交互的，访客点开就能运行。

发帖编辑器：
```
[📸 附件] [🎬 短视频] [📊 表格] [🔗 嵌入仿真]
                                    ↑
                                 核心入口

用户点击后：
  ├─ [从我的库导入] - 用户之前保存的 Wokwi 项目
  ├─ [创建新仿真] - 从头开始搭建电路
  ├─ [从剪贴板粘贴代码] - 快速导入代码
  └─ [Fork 蓝图库项目] - 基于官方蓝图修改

效果：帖子中嵌入完整的 Wokwi 仿真器，其他用户可以：
- 直接运行代码看效果
- Fork 到自己的工作台
- 在线修改和调试
- 实时评论和建议
```
