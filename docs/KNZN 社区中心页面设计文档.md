# 👥 KNZN 社区中心 (Community Hub) v1.0 - 完整设计文档

## 🎯 核心定位

**社区中心 (Community Hub)** 是 KNZN 平台的**社交和知识共享核心**，用户在这里可以：

- 📝 分享硬件项目、设计心得、焊接技巧
- 💬 讨论蓝图、提问求助、解答他人疑惑
- 🏆 展示成就、参与竞赛、获得认可
- 🤝 寻找志同道合的开发者、建立学习小组
- 📢 发布赛事、工作坊、线下活动
- 🎓 分享学习资源、推荐教程、交流最佳实践

社区中心不仅是论坛，更是**学习型社交网络**，驱动用户留存、内容生成、变现机会。

## 🎬 硬件社区特有的四大功能模块（v1.1 新增）

KNZN 社区中心相比通用平台（Reddit、Discord）的**竞争壁垒**在于以下四个硬件垂直的核心功能：

### 1️⃣ 🎬 富媒体调试沙箱 (Embedded Debugging Sandbox)

传统社区（Reddit/Discord）的讨论只支持**文字+图片+代码片段**。但硬件问题需要**实时可运行的证据**：

```
❌ 仅贴代码："第 10 行 delay() 有问题" → 大神无法复现
✅ 嵌入仿真："见下方仿真，LED 在 T=0.5s 时出现毛刺" → 可立即调试

❌ 仅贴原理图截图："这是 L298N 电路" → 无法验证设计
✅ 嵌入仿真模型："这是仿真运行结果，电流溅射出现了" → 可迭代修复
```

**解决方案：Wokwi + HardwareSimulator 集成**

发帖编辑器增强：
```
[📸 附件] [🎬 短视频] [💻 代码] [🔗 链接] [📊 表格]
[+ 嵌入仿真] ← 新增快捷入口

用户点击后：
  ├─ [从我的库导入] - 用户之前保存的仿真
  ├─ [创建新仿真] - 从头开始
  ├─ [从剪贴板粘贴代码] - 快速导入
  └─ [仿真预览] - 实时显示运行效果
```

### 2️⃣ 🏭 打板拼车广场 (Fabrication Carpooling Square)

硬件制造有天然的**成本障碍**：

```
PCB 打样：通常 5 片起订
         小白只想要 1 片 → 被迫买 5 片 → 成本 ¥50-100 浪费

芯片采购：最小包装 (MOQ) 通常 10-50 个
         学生项目只需 3 个 → 要买 10 个 → 浪费 70%

3D 打印：同一个设计需要 6-8 小时
         多个小订单不如一次集中打印 → 时间浪费
```

**解决方案：拼单广场**

拼单广场首页：
```
筛选：[全部 ▼] [PCB] [芯片] [3D打印] [焊接]
排序：[最新] [最快成团] [成本最低]

🟢 L298N 电机驱动双轴板
   发起者：John Smith (Level 15, ✅ 信誉好)
   拼单进度：2/3 人已加入
   成本分摊：¥32/人 (vs 单独打样 ¥60)
   预计交付：2025-01-05
   还需：1 人  [我要加入]
```

### 3️⃣ ✅ 问答采纳系统 (Accepted Answer & Solved Status)

目前的设计只有**"点赞"**。但在技术问答中：

```
❌ "点赞最多" ≠ "解决了问题"
❌ 新用户看到高赞回复，但可能过时或无关
❌ 提问者无法标记"这个解决了我的问题"
❌ 高质量回答者没有额外奖励
❌ 没有"已解决"的标记，后人重复提问
```

**解决方案：StackOverflow 采纳模式**

被采纳的回答者获得：
- ✅ +50 XP （vs 普通回答 +30 XP）
- ✅ 一枚"最佳答案"小徽章
- ✅ 回答被固定在帖子顶部
- ✅ 影响"帮助者排行榜"排名

### 4️⃣ 🔗 蓝图关联系统 (Blueprint Integration)

用户发帖"晒项目"时，**缺乏与蓝图库的结构化连接**。

**解决方案：蓝图关联元数据**

发帖编辑器增强：
```
🔗 关联蓝图（可选，但推荐）

✅ 已关联：
┌─ L298N 双轴电机驱动 v2.1 ✅ ──────────────────────────────┐
│ 版本：v2.1                                                │
│ 状态：此蓝图的基础上修改                                   │
│ [修改关联] [移除]                                          │
└──────────────────────────────────────────────────────────┘

关联关系（用户可选）：
  ⦿ 原样不动使用 (No Modification)
  ⦿ 基础上修改 (With Modifications)
  ⦿ 灵感来自 (Inspired By)
```
